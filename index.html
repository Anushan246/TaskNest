Task Management

Organize your day efficiently and easily!

How to use:

  • Type your task, select a priority, and click Add Task.
  • Mark a task as Important and set a password to protect it.
  • Click a task to mark it as completed.
  • Click Delete to remove a task (password required for important tasks).
  • Use Delete All to clear all tasks.
  • Filter tasks by priority using the filter buttons.
